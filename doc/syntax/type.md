# 类型

## 基础类型

```do
i8    // 8 位有符号整数
i16   // 16 位有符号整数
i32   // 32 位有符号整数
i64   // 64 位有符号整数
u8    // 8 位无符号整数
u16   // 16 位无符号整数
u32   // 32 位无符号整数
u64   // 64 位无符号整数
isize // 指针宽度有符号整数
usize // 指针宽度无符号整数
f32   // 32 位浮点数
f64   // 64 位浮点数
bool  // 布尔类型
text  // 文本类型
```

## nil

```do
done() -> nil {
    return // nil 返回
}

value User | nil = nil // 可空值绑定
```

## 连续存储

```do
[u8]         // 字节连续存储
[i32]        // i32 连续存储
[User]       // 结构体连续存储
[User | nil] // 可空元素连续存储
[[u8]]       // 嵌套连续存储
```

## 命名类型

```do
User              // 命名类型
Box<i32>          // 单参数泛型类型
Entry<text, User> // 多参数泛型类型
Map<text, [User]> // 泛型参数内使用连续存储
```

## 函数类型约束

```do
#F = () -> nil          // 无参数 nil 返回函数类型约束
#G = (i32) -> i32       // 单参数单返回函数类型约束
#H = (i32, text) -> bool // 多参数单返回函数类型约束
#P = (i32) -> i32, bool // 单参数多返回函数类型约束
```

## 类型别名

```do
MaybeUser = User | nil       // 可空联合别名
ReadResult = [u8] | FileError // 值或错误联合别名
```
