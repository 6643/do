# do fmt 07.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a first-version `do fmt` command with stable formatting output and regression coverage.

**Architecture:** Keep formatting in `tool/fmt/` with a pure formatter plus a small command runner. Do not reuse the compiler lexer as the only input because it drops whitespace and comments; build a formatter scanner that preserves trivia and emits deterministic bytes. Wire the command through existing CLI patterns and test it through `tool/build/test/run_tests.sh`.

**Tech Stack:** Zig compiler code, shell regression harness, `.do` fixture files, Markdown docs.

- [x] **Step 3: Implement `FmtArgs` and `parseFmt`**
  - 已完成: `FmtArgs` 支持 `input_path` 和 `check`，`parseFmt` 只接受一个输入文件和可选 `--check`。

Add:

```zig
pub const FmtArgs = struct {
    input_path: []const u8,
    check: bool = false,
};

pub fn parseFmt(args: []const []const u8) !FmtArgs {
    var input_path: ?[]const u8 = null;
    var check = false;
    var i: usize = 1;
    while (i < args.len) : (i += 1) {
        if (std.mem.eql(u8, args[i], "--check")) {
            if (check) return error.UnexpectedCliArg;
            check = true;
            continue;
        }
        if (std.mem.startsWith(u8, args[i], "-")) return error.UnexpectedCliArg;
        if (input_path != null) return error.UnexpectedCliArg;
        input_path = args[i];
    }
    return .{
        .input_path = input_path orelse return error.MissingInputPath,
        .check = check,
    };
}
```

```zig
const cli = @import("../build/cli.zig");

pub fn run(init: std.process.Init, args: []const []const u8) !void {
    const allocator = init.gpa;
    const io = init.io;
    const parsed = cli.parseFmt(args) catch |err| {
        try diag.printCliError(io, err);
        std.process.exit(1);
    };
    _ = allocator;
    _ = io;
    _ = parsed;
    return error.FmtNotImplemented;
}
```
